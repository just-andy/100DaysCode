Just Andy's 100 Days of Code

Last Updated: 1st Jan 2021

2020 was hard for everyone in several different ways. Although I was fortunate enough to still be employed I did find frustations towards the end of the year.

I identified that I had lost the mental stimulation of learning. Although I work in code as part of my day job I wasn't taking the time to learn all the courses or tutorials I'd bookmarked. I took Wes Bos "Master Gatsby" course and realised the challenge of learning and problem solving was what I needed for my mental health.

This loooong list is my notes on what I tackled on my #100DaysofCode.

Day 6 - 1st January 2021

Really short one today as I mainly focused on getting this site together. Went for straight HTML and Tailwind to get up and running.

I played around with Github pages but realised it was overkill for what I needed.

Day 5 - 31st December 2020

I've been having trouble making a React Navigation with multiple dropdowns. To focus my efforts I set up a codesandbox to get focus on getting the issue. Still needs finished off but its a start. Watched tutorials around React useRef from Web Development Made Simple.

Create a new git repo for CRA, Tailwind and Wes Bos EsLint code for Future projects that I can fork and start from.

Day 4 - 30th December 2020

Refactored a lot of Gatsby code on our Design System website.

Included Wes Bos's ES Lint and Prettier on the project to start cleaning up the bad coding. I've found this approach really useful after using the setup in his courses.

Day 3 - 29th December 2020

Gatsby/Netlify/Github deploy a client site and deal with DNS issues. Learnt that I should always remember to DOUBLE CHECK DNS settings. You can check out the salon site here at chapteronehairspa.co.uk.

Lesson 3 of Wes Bos JS 30, learning about CSS Variables .

Read when and why of every/forEach in Javascript. Very useful breakdown. - When and Why to use the .every() Array Method in Javascript.

Day 2 - 28th December 2020

Short and simple one, finished the last few lessons on Wes Bos "Master Gatsby".

Really great course, can't recommend it enough. Most other Gatsby courses cover the basics but this goes to a higher level working with Sanity, Netlify functions and more.

Day 1 - 27th December 2020

Finishing the last few lessons Master Gatsby JS, specifically on generating dynamic pages.

Checked out Jason Lengstorf and Ben Ilegbodugoing through the basics of React and Typescript. This was a great episode that covered set up and the basics with examples. TS is something I really hope to get to grips with in these 100 days so it was a great primer.
